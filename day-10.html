01 / The Canvas

Columns, Rows, and the fr unit

Flexbox is for one-dimensional layouts (rows OR columns). Grid is for two-dimensional layouts (rows AND columns). The magic of Grid lies in the fr (fractional) unit, which distributes available free space automatically.

/* The fr unit calculates space AFTER fixed tracks (like px or rem) are resolved */
.layout {
  display: grid;
  gap: 20px;
  grid-template-columns: 250px 1fr 2fr; /* Fixed, flexible, extra flexible */
  grid-template-rows: auto 1fr auto;    /* Fits content, stretches, fits content */
}

03 / The Holy Grail of Responsiveness

auto-fill vs auto-fit

Combined with minmax(), these two keywords allow you to create fully responsive, wrapping grids without writing a single media query. But understanding their difference is crucial.

auto-fill (leaves empty tracks)

Item 2
Item 3
Fills the row with as many 80px tracks as possible. If there aren't enough items, the empty tracks still take up space.
auto-fit (collapses empty tracks)
Item 1
Item 2
Item 3
Also creates tracks, but collapses empty ones to 0px, causing the existing items to stretch (because of 1fr) and fill the space. 
/* The ultimate responsive card grid formula */
.card-grid {
  display: grid;
  gap: 1.5rem;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
}
04 / The Traps

Common gotchas

⚠️
Percentages vs the fr unit
❌ Using % with gap causes overflow
50%
50%
50% + 50% = 100%. Add a 10px gap, and the grid is now 100% + 10px wide, causing a horizontal scrollbar.
✓ Use fr units
1fr
1fr
The fr unit calculates space after gaps are subtracted. No math required, no overflow.
⚠️
Grid blowout (min-width: auto)
Just like Flexbox, Grid items have an implicit min-width: auto. If you put a wide image or preformatted text inside a grid cell, it might stretch the entire track and break your layout instead of wrapping or shrinking.

✓ The Fix
Add min-width: 0 to the grid child to allow it to shrink below its content's intrinsic size. (For rows, use min-height: 0).
05 / Key Points

Grid Foundations.

1fr > %

Never use percentages for tracks if you are using gap. The fr unit handles the math for you automatically. grid-cols-2

repeat() saves typing

Use repeat(12, 1fr) instead of writing out twelve tracks. You can repeat single tracks or complex patterns.

auto-fit = responsive

repeat(auto-fit, minmax(250px, 1fr)) is the most powerful line in CSS Grid. It builds robust, wrapping card grids with zero media queries.

auto-fit vs auto-fill

Fill leaves empty tracks at the end of the row. Fit collapses those empty tracks, letting the existing items stretch.
