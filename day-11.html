Grid Placement & Areas

Mastering the two-dimensional canvas. Moving beyond auto-placement to explicitly control where elements live, how they span, and how they overlap.

01 / Explicit Placement

Line-Based Placement & Spanning

Grid creates numbered lines between your tracks. In a 3-column grid, there are 4 vertical lines (1, 2, 3, 4). You can tell any item exactly which lines to start and end on.

/* Start at line 1, end at line 3 */
.box-1 { grid-column: 1 / 3; }

/* Start at line 2, span across 3 tracks */
.box-2 { grid-column: 2 / span 3; }

/* 💥 THE MAGIC TRICK */
/* -1 always targets the very last line */
.full-width {
  grid-column: 1 / -1;
}

Pro-tip: Using 1 / -1 is the safest way to break an item out of a column structure to span the full width, regardless of how many columns the grid currently has.

02 / ASCII Art CSS

grid-template-areas

Line numbers get confusing in complex layouts. grid-template-areas lets you visually "draw" your layout using named strings. It is the most readable way to build app dashboards and article layouts.

header
nav
main
aside
footer 
.dashboard {
  display: grid;
  grid-template-areas:
    "header header header"
    "nav    main   aside"
    "nav    footer footer";
}

.sidebar { grid-area: nav; }
.content { grid-area: main; }
/* Assign grid-area to elements to slot them in! */

To leave an empty space in the layout, use a dot "." in your string.

03 / The Masonry Fix

grid-auto-flow: dense

By default, if an item is too wide to fit the remaining space in a row, Grid pushes it to the next row, leaving a hole. The dense packing algorithm tells Grid to go back and fill those holes with smaller items that come later in the DOM.

1
2
3 (Too wide, moves down)
4 (Small enough to backfill!)
5
6
.photo-gallery {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
  grid-auto-flow: row dense; /* Backfills the holes! */
}
04 / Breaking the Box

Overlapping without Absolute Positioning

Before CSS Grid, layering elements required position: absolute, which removes elements from the document flow (making parent containers collapse). With Grid, you can assign multiple items to the exact same grid cells. They remain in the flow, and you can control layering with z-index.

Grid Layering

This text box is sitting inside the same grid as the background image. No absolute positioning required. The parent perfectly contains both.

NEW
.card {
  display: grid;
  grid-template-columns: repeat(6, 1fr);
}

.background-img {
  grid-column: 1 / -1;
  grid-row: 1 / -1; /* Covers the whole grid */
}

.text-box {
  grid-column: 2 / 6; /* Sits precisely in the middle */
  grid-row: 2 / 4;
  z-index: 10;    /* Sits on top */
}
