05 / Key Points

Advanced flex patterns.

margin: auto = spacer

Auto margins consume remaining flex space on that side. The cleanest way to push elements to opposite ends. ml-auto mt-auto

min-width: 0

When a flex child won't shrink — add min-width: 0. Default is auto (content size). Also required for text-overflow: ellipsis to work. min-w-0

Images need align-self

align-items: stretch distorts images. Add align-self: flex-start to images (or the container). self-start

flex-wrap for responsive

flex-wrap: wrap + flex: 1 + min-width gives you a responsive grid without media queries. The simplest responsive layout pattern. flex-wrap
